[3] What is the free store? What other name is commonly used for it? What operators support it?
The free store (AKA heap) is available through the operator new. It is any memory not in the other three categories of memory and therefore potentially "free":

code or text storage	|	memory set aside by compiler for the code on program start
static storage	|	memory set aside by compiler for the global variables on program start
stack or automatic storage	|	memory set aside by compiler on function calls, containing arguments and local variables

[4] What is a pointer?
A pointer value is the address of the first byte of the memory.
A pointer points to an object of a specified type.
A pointer does not know how many elements it points to
