[7] What is an STL iterator? What operations does it support?
indicates the beginning and end of a sequence

p==q		true if and only if both point to the same element or both point to one beyond the last element
p!=q		!(p==q)
*p		element pointed to by p
*p=val		writes to element pointed to by p
val=*p		reads from element pointed to by p
++p		makes p refer to the next element
